### fin!

#### · 구현 기능
- [x] 이름 입력 후 해당 내용 local storage에 저장
- [x] 현재 시간 (좌측 상단에 위치)
- [x] 위치해 있는 지역과 그 지역의 날씨 (우측 상단에 위치)
- [x] 랜덤 배경
- [x] 랜덤 명언 (하단에 위치)
- [x] 투두 리스트 <br />
  → 투두 리스트 입력 후 해당 내용 local storage에 저장 <br />
  → 투두 리스트 삭제 시 화면 및 local storage에서도 삭제
